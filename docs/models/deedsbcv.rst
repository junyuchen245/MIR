deedsBCV
========

Overview
--------
MIR integrates the deedsBCV binary for baseline registration on Linux.

Notes
-----
- Linux-only support.
- Binaries can be bundled or downloaded via Google Drive if configured.

Key modules
-----------
- ``MIR.models.deedsBCV``
- ``MIR.utils.deedsbcv_binary``
